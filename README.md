Speech-to-Text Web

A simple web application that converts speech/audio files into text using OpenAI Whisper, built with Flask and deployed via Ngrok (tested on Google Colab).

Features
- Record voice or upload audio
- Automatic audio conversion (FFmpeg)
- Accurate speech-to-text using Whisper
- Clean UI with Flask

 Tech Stack
Python, Flask, HuggingFace Transformers, Whisper, Torchaudio, FFmpeg
